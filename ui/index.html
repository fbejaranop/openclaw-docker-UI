OpenClaw container launcher

Choose only the providers you need, then launch a new Dockerized OpenClaw instance from this page.

Model providers

Only the selected providers are required and written into the generated config.

Container tools

These are baked into the image when the container is created.

SSH setup

Host mode mounts your machine SSH folder read-only and copies its files into the instance on first boot, so the container does not edit the originals.

Slack profiles are saved in this browser with nickname, bot token, app token, and allowed channel ID.

Slack app manifest preview
This is generated from the instance name, so you can create the Slack app from scratch without guessing the config.
